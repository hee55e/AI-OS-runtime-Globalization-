# Primary Core 기본 세트

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.
> 참고: 권장 예시 이름은 아래 UTCP 매뉴얼의 실제 `name`을 기준으로 사용해야 합니다.

## 현재 구현
- 기본 세트는 “도구 다량 제공”보다 “작은 코어 번들 + 필요 시 확장” 전략이 맞습니다.
- 실제 바인딩은 `ProcessorUnit.allowed_tools`에 UTCP 매뉴얼 `name`을 넣는 방식입니다.
- 권장 코어는 파일 5개 + 실행 4개 + 검색 1개 + 그래프/도구관리 소수로 시작합니다.

## 프론트엔드 연결
- 도구 바인딩 편집은 Processor Editor와 Tool Builder 플로우에서 수행합니다.
- AI Explorer에서 시스템 도구/사용자 도구를 함께 탐색할 수 있습니다.

## 백엔드 연결
- 도구 매뉴얼 조회: `kait_os/routers/tools/utcp_tools.py`
- 사용자 도구 CRUD: `kait_os/routers/tools/user_tools.py`
- 바인딩 변환: `kait_os/core/execution/tool_binding_handler.py`
