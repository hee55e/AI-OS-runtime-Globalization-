# TTS 다이렉트 (텍스트 → 음성)

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- TTS Direct provider는 `generic`, `elevenlabs`, `gemini`, `qwen`, `placeholder`가 존재합니다.
- 채팅 응답 자동 재생 옵션은 사용자 설정(`autoPlay`)을 따릅니다.
- voice stream API의 TTS 엔드포인트(`/api/v1/voice/stream/tts`)를 통해 합성 결과를 전달합니다.

## 프론트엔드 연결
- Settings > Voice 탭에서 TTS provider/model/voice 설정
- Chat/Voice UI에서 자동 재생 on/off 사용

## 백엔드 연결
- TTS providers: `kait_os/core/tts_direct/providers/`
- Provider schema: `/api/v1/providers/tts/*`
- TTS 스트림 엔드포인트: `kait_os/routers/conversation/voice_stream.py`
