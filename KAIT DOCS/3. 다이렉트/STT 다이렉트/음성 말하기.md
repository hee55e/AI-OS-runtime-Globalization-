# STT 다이렉트 (음성 → 텍스트)

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- STT Direct provider는 `generic`, `whisper`, `elevenlabs` 구현체가 등록되어 있습니다.
- 음성 스트림 파이프라인에서 chunk 업로드 후 텍스트 변환 단계에 사용됩니다.
- Provider 스키마 API로 모델 목록을 동적으로 제공받아 설정 UI를 구성합니다.

## 프론트엔드 연결
- 음성 설정 탭에서 STT provider/model 선택 가능
- 음성 훅(`useVoiceChat`)이 스트림 API를 호출합니다.

## 백엔드 연결
- STT providers: `kait_os/core/stt_direct/providers/`
- Provider schema: `/api/v1/providers/stt/*`
- 음성 스트림 라우터: `kait_os/routers/conversation/voice_stream.py`
