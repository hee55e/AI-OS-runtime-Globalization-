# 디버그

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- Debug Panel은 채팅 블록을 Trace로 재구성해 Timeline/Inspector로 시각화합니다.
- Graph/Flow 뷰 전환, 노드 선택 상세, 스트리밍 이벤트 확인을 지원합니다.
- 실행 로그는 WAL 기반으로 기록되며 정상 완료 시 정리됩니다.

## 프론트엔드 연결
- 디버그 패널: `components/debug/DebugPanel.tsx`
- 스트림 워크벤치: `components/debug/workbench/StreamWorkbench.tsx`

## 백엔드 연결
- WAL 기록기: `kait_os/services/execution_logs/wal_writer.py`
- ExecutionState 업데이트: `kait_os/services/execution/execution_runner.py`
