# 음성 설정

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- 음성 기능은 STT/TTS provider 스키마를 동적으로 받아 모델/보이스 선택을 구성합니다.
- 실시간 음성 흐름 API는 stream 모드(`/api/v1/voice/stream/*`)와 RTC 모드(`/api/v1/voice/rtc/*`)를 제공합니다.
- 사용자 음성 설정은 executor-config의 custom settings에 저장됩니다.

## 프론트엔드 연결
- 설정 탭 음성 UI: `UserSettingsModal.tsx`
- 음성 전용 모달: `VoiceSettingsModal.tsx`
- 음성 훅: `hooks/useVoiceChat.ts`

## 백엔드 연결
- Provider 스키마: `/api/v1/providers/stt/*`, `/api/v1/providers/tts/*`
- 스트림 엔드포인트: `kait_os/routers/conversation/voice_stream.py`
- RTC 엔드포인트: `kait_os/routers/conversation/voice_rtc.py`
