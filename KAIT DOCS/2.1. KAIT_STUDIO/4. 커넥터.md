# 커넥터

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- 커넥터는 native / 외부 OAuth / custom MCP를 통합 관리하는 레이어입니다.
- 사용 가능한 커넥터 정의는 `kait_os/connectors/manifest.json`이 단일 소스입니다.
- 연결 후 도구 등록(등록/해제)까지 API에서 일괄 처리됩니다.

## 프론트엔드 연결
- 커넥터 모달: `components/connectors/ConnectorsModal.tsx`
- native 설정 패널: FileSystem / Terminal / Messenger 패널

## 백엔드 연결
- 커넥터 CRUD/도구 연동: `kait_os/routers/connectors/connectors.py`
- OAuth 플로우: `kait_os/routers/connectors/connector_auth.py`
- 자격증명/토큰 저장: `kait_os/routers/connectors/connector_config.py`
