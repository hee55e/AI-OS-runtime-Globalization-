# 설정

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- 설정 진입점은 ActivityBar 하단 Settings 버튼이며 모달 탭 구조로 구성됩니다.
- 탭: profile / env / executor / voice / segments / appearance / performance
- 설정 저장은 대부분 즉시 API 반영이며 일부 UI 테마는 로컬 스토어와 함께 유지됩니다.

## 프론트엔드 연결
- 설정 모달: `kait_os/kait_studio/src/components/settings/UserSettingsModal.tsx`
- 성능 패널: `kait_os/kait_studio/src/components/settings/PerformanceSettingsPanel.tsx`

## 백엔드 연결
- 사용자 환경 변수: `/api/v1/users/me/env`
- 실행기 설정: `/api/v1/users/me/executor-config/*`
- Provider 스키마: `/api/v1/providers/*`
