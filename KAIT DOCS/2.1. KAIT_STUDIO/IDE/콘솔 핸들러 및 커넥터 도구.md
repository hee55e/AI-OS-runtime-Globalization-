## 현재 구현
- 터미널 관련 핵심 UTCP 도구는 `terminal`, `run_command`, `command_status`, `kill_command`입니다.
- 커넥터 설정으로 Terminal 접근 정책(`TERMINAL_CONFIG`, `SHELL_CONFIG`)을 사용자별 저장할 수 있습니다.
- 현재 터미널 REST 라우터는 mock user dependency가 남아 있어 auth 연동 TODO가 있습니다.
