# 리플레이 재현

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- 완전한 실행 리플레이(타임라인 재생) UI는 현재 구현되어 있지 않습니다.
- 대신 디버그 패널 타임라인, WAL 실행 로그, 터미널 reconnect replay가 부분 기능으로 존재합니다.
- 앱 뷰포트는 버전 복원 도구(`restore_app_version`)를 통해 부분 롤백이 가능합니다.

## 프론트엔드 연결
- 디버그 패널 타임라인: `components/debug/DebugPanel.tsx`
- 스트림 워크벤치: `components/debug/workbench/StreamWorkbench.tsx`

## 백엔드 연결
- WAL 기록기: `kait_os/services/execution_logs/wal_writer.py`
- 터미널 replay 이벤트: `kait_os/routers/execution/terminal.py`
- 앱 버전 복원 도구: `kait_os/tools/handlers/viewport/app/tools.py`
