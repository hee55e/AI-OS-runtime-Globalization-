# 컨텍스트 필터: 워크스페이스

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- `workspace_filter`는 현재 작업 경로/파일 상태 요약을 시스템 프롬프트에 주입합니다.
- 워크스페이스 스캔/검색 도구는 `ai_builder/workspace_manager` 카테고리 UTCP로 제공됩니다.
- 대용량 워크스페이스는 요약 위주로 전달하고 상세 조회는 도구 호출로 분리하는 구조입니다.

## 프론트엔드 연결
- File Explorer와 VFS/로컬 탐색 UI에서 현재 경로 맥락을 제공합니다.

## 백엔드 연결
- 필터 구현: `kait_os/core/context/filters/system_prompt_parts/base/workspace_filter/`
- 워크스페이스 도구: `kait_os/utcp/categories/ai_builder/workspace_manager/*.json`
- 파일 브라우저 API: `kait_os/routers/data/file_browser.py`
