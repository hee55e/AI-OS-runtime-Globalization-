# 컨텍스트 필터: 프로젝트

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- `project_filter`는 프로젝트 메타(이름/설명/구조 요약)를 시스템 프롬프트로 구성합니다.
- 그래프/프로세서/도구 구성 정보와 함께 프로젝트 단위 인지 맥락을 보강합니다.
- 프로젝트 전체를 본문으로 넣기보다 요약 + 필요 시 조회 도구 호출 패턴을 권장합니다.

## 프론트엔드 연결
- 프로젝트 드롭 열기/앱 렌더러는 Viewport App 모드에서 처리됩니다.

## 백엔드 연결
- 필터 구현: `kait_os/core/context/filters/system_prompt_parts/base/project_filter/`
- 프로젝트 탐색 도구는 workspace_manager + filesystem 도구 조합을 사용합니다.
