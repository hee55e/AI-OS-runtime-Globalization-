# 갤러리 이용하기

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- 갤러리는 생성/업로드된 미디어를 통합 조회하는 레이어이며 현재 image/video 중심입니다.
- 서버 API는 `/api/v1/gallery/items`, `/stats`, `/sources`, `DELETE /items/{id}`를 제공합니다.
- 클라이언트는 필터/검색/다중선택/삭제/업로드/이미지 편집/AI 질의 흐름을 지원합니다.
- 실시간 이벤트(`asset_created`, `asset_deleted`)를 구독해 목록을 즉시 동기화합니다.

## 프론트엔드 연결
- 메인 모달: `components/gallery/GalleryModal.tsx`
- 편집 모달: `components/gallery/ImageEditModal.tsx`

## 백엔드 연결
- 갤러리 라우터: `kait_os/routers/data/gallery.py`
- 이미지/비디오 저장소: `kait_os/modules/image_manager/services/storage.py`, `video_manager/services/storage.py`
