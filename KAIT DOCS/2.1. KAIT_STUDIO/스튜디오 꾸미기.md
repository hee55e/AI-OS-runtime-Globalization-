# 스튜디오 꾸미기

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- 테마/텍스처/강조색은 스토어와 CSS 토큰 변수 기반으로 적용됩니다.
- 세그먼트 템플릿/스타일은 executor-config API로 사용자 단위 저장/프리셋 관리가 가능합니다.
- 그래프 시각화는 기본 뷰 + 신호 흐름 시각화 컴포넌트를 포함합니다.

## 프론트엔드 연결
- 테마 훅: `hooks/useThemeVariables.ts`
- 텍스처 토큰: `styles/tokens/textures/*`
- 세그먼트 설정: `components/settings/SegmentTemplateSettings.tsx`

## 백엔드 연결
- 세그먼트 템플릿/스타일 API: `kait_os/routers/system/user_executor_config.py`
- 세그먼트 레지스트리: `kait_os/core/common/segment_registry/`
