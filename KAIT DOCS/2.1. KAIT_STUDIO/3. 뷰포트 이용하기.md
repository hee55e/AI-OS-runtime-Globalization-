# 뷰포트 이용하기

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- Studio는 인스턴스 기반 탭 뷰포트(`viewportStore`)로 동작하며 기본 탭은 `main_graph`입니다.
- 프론트 타입: graph / browser / file / terminal / preview / app / corpus
- 백엔드 슬롯 manifest: artifact / corpus / electron_browser / file / preview / terminal
- 브라우저/터미널 등 일부 뷰포트는 백엔드 세션 또는 Electron 브리지와 연동됩니다.

## 프론트엔드 연결
- 뷰포트 컨테이너: `components/viewport/UniversalViewport.tsx`
- 스토어: `stores/viewportStore.ts`
- 슬롯 렌더러: `components/viewport/slots/*`

## 백엔드 연결
- 뷰포트 슬롯 레지스트리: `kait_os/core/viewport/registry.py`
- 슬롯 manifest: `kait_os/core/viewport/slots/*/manifest.json`
- 브라우저 제어: `kait_os/routers/viewport/electron_browser.py`
- 세션 라우터: `kait_os/routers/viewport/session.py`
