> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- 핵심 런타임은 FastAPI/SQLAlchemy/Pydantic 중심이며 도구/멀티모달/오케스트레이션 모듈이 결합됩니다.
- requirements에는 `langchain`, `langgraph`가 포함되어 있지만 현재 코드 import는 확인되지 않습니다.
- google-genai SDK 관련 알려진 경고(`TrafficType`)는 provider에서 warning suppression 처리되어 있습니다.
