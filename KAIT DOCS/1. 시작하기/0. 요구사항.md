# 시스템 요구사항

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- 백엔드는 FastAPI + SQLAlchemy + Pydantic v2 구조이며 메인 엔트리는 `kait_os/app.py`입니다.
- 인증/세션/실행 상태는 DB에 저장되며 기본 로컬 실행에서는 SQLite를 사용합니다.
- Memory System(임베딩 인덱싱)은 `MEMORY_ENABLED` + Redis(arq) + ChromaDB가 있을 때 활성화됩니다.
- 데스크톱 스튜디오는 React(Vite) 기반이며 Electron 브리지 기능은 Electron 실행 시에만 동작합니다.

## 프론트엔드 연결
- 앱 진입: `kait_os/kait_studio/src/App.tsx`
- 셸 레이아웃: `kait_os/kait_studio/src/components/shell/AppShell.tsx`

## 백엔드 연결
- API 라우터 등록: `kait_os/app.py`
- DB 모델: `kait_os/db_models/`
- 메모리 인덱싱 훅: `kait_os/core/memory/indexing_hooks.py`
