## 현재 구현
- 현재 코드에는 독립된 온보딩 위저드(quick/manual 분기)가 구현되어 있지 않습니다.
- 실제 초기 설정은 로그인 후 User Settings에서 수동으로 진행합니다(Env/Executor/Voice/Appearance).
- API 키/실행 정책/세그먼트 스타일은 각각 `user_env`, `user_executor_config` API로 저장됩니다.
