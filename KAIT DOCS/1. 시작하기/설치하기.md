# 설치 및 실행

> 코드 동기 기준: 2026-02-03
> 원칙: 문서보다 코드가 우선입니다.

## 현재 구현
- 코드 기준으로는 개발 실행 절차가 우선이며 인스톨러 배포 절차는 별도 문서/스크립트로 분리되어 있습니다.
- 백엔드는 `kait_os`에서 Python 엔트리(`main.py` 또는 platform 모듈)로 실행합니다.
- 예제 UI/CLI 실행 스크립트는 루트의 `run_web_ui.bat`, `run_cli_chat.bat`를 사용합니다.

## 프론트엔드 연결
- 웹 UI 실행 보조 스크립트: `run_web_ui.bat`
- 데스크톱 개발 실행 보조 스크립트: `run_desktop_dev.bat`

## 백엔드 연결
- 백엔드 실행 예시: `kait_os/main.py`, `kait_os/platforms/cloud/main.py`
- 빠른 시작 참고: `QUICK_START.md`
