# KAIT 설치하기 (Installation) (미정)

**KAIT**는 복잡한 설정 없이 간편하게 설치할 수 있는 전용 인스톨러를 제공합니다.

## Windows 설치

1. **다운로드**
   [공식 다운로드 페이지](#)에서 최신 버전의 `KAIT-Studio-Setup-x.x.x.exe` 파일을 다운로드합니다.

2. **설치 진행**
   다운로드한 파일을 더블 클릭하여 실행합니다.
   *   *팁: Windows 보안 경고(SmartScreen)가 뜰 경우 '추가 정보' -> '실행'을 클릭하세요.*

3. **구성 요소 선택**
   *   **Standard (기본)**: KAIT Studio와 필수 Kernel만 설치합니다. (권장)
   *   **Full (전체)**: 로컬 LLM 구동을 위한 추가 라이브러리(CUDA 등)를 포함합니다. 용량이 큽니다.

4. **완료**
   설치가 완료되면 바탕화면의 `KAIT Studio` 아이콘이 생성됩니다.
